eBay plugin

This plugin reads from eBay orders to automatically fill in the receiver details in freight bookings. It will also pass the sku values through to any inventory plugin that might be loaded.
Go into eBay and export the orders as csv. Save the file in the IoTKeys application directory data/ebay. You can have multiple ebay sites, so there can be multiple files in the data/ebay directory.
So if you installed IoTkeys into c:\apps\IoTkeys then the file should be saved in c:\apps\IoTkeys\data\ebay
The plugin can only see the orders that have been exported. So the files need to be up to date. When all the orders have been processed then the file can be removed. Storing too many order files will slow down the processing with no real benefit.
