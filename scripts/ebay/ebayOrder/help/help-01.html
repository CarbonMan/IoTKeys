eBay plugin

This plugin reads from eBay orders to automatically fill in the receiver details in freight bookings. It will also pass the sku values through to any inventory plugin that might be loaded.
Go into eBay and export the orders as csv. Save the file in the IoTKeys application directory as data/ebayOrders.csv
So if you installed IoTkeys into c:\apps\IoTkeys then the file should be saved as c:\apps\IoTkeys\data\ebayOrders.csv
The plugin can only see the orders that have been exported. So the file needs to be up to date.
