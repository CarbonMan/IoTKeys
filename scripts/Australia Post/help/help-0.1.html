A plugin to making Australia Post shipments

This plugin can connect with order plugins, weighing and cubing devices and an inventory system. The diagram below shows the process but it is really simple.
  1. The Australia Post login screen will be brought up
  2. Log in (don't worry NONE of the processing happens on our servers)
  3. Enter a reference number (ie. an eBay order number)
  4. If order plugins are detected they are queried for the reference number
  5. The order plugin returns the receiver details and they are pushed into the booking
  6. If devices are detected they are queried for values. Weighing scales and cubing machines are handled.
  7. The device result are pushed into the booking
  8. You accept the booking
  9. If an inventory plugin is detected the order skus and quantities are passed in
  10. The inventory system can adjust the stock levels
