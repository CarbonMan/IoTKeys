wooCommerce plugin

This plugin reads from wooCommerce orders to automatically fill in the receiver details in freight bookings. It will also pass the sku values through to any inventory plugin that might be loaded.
Go into your wooCommerce interface and export the orders as json. Save the file in the IoTKeys application directory as data/wooCommerce.json
So if you installed IoTkeys into c:\apps\IoTkeys then the file should be saved as c:\apps\IoTkeys\data\wooCommerce.json
The plugin can only see the orders that have been exported. So the file needs to be up to date.
