SF-884 is a weigh scales indicator


If you are using this device without any other configuration options then the weight readings are available by pressing and holding the Ctrl and Alt keys then tapping the w key. This will work regardless of the application.

CONFIGURATION

The configuration details are found in 2 files
  1. keyMaps.json
  2. devices.json
These are described below. If, for any reason, changes are made in these files then the contents should be validated here;
https://jsonlint.com/
If the structure of the files is invalid then the application may not start correctly.

keyMaps.json

This sets what key combination will read the scales. There may be situations where a key combination could interfere with the active application. In these cases the key combination can be changed.
  • alt - Whether the Alt key needs to be pressed (false/true)
  • ctrl - Whether the Ctrl key needs to be pressed (false/true)
  • keyCode - The keypress code to look for. A full list of keycodes is available here
    https://raw.githubusercontent.com/CarbonMan/IoTKeys/master/config/KeyCodes.txt

devices.json

Usually the only value that should be changed is the portName. If set to "?" then this will cause IoTKeys to scan the ports on startup. This may not work in all cases. In situations where the scan detects the wrong USB device then the correct port needs to be entered into this file. If the device is moved to another port then it will have to be reentered. To set the COM port first you have to determine the port number. On Windows 8/10 right click the start button

The COM port is indicated in the (). So in this case it is "COM4". The portName line in the devices.json file would be: ... "portName" : "COM4", ...

Troubleshooting

If the SF-884 device is shown in the Ports with a warning, that will usually indicate that a device driver needs to be installed. You will need to obtain administrative privileges from your network or IT administrator in order to install these drivers. The driver can be found here https://github.com/CarbonMan/IoTKeys/raw/master/config/devices/CH341SER.EXE
