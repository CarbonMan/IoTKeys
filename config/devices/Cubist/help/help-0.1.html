Cubist is a cubing device

Instantly detect the cubic dimensions of a parcel. Works with other plugins to automatically insert L x W x H into web pages.
Cubist uses a sensor to detect cubic dimensions.

Installation

An additional step is required to get the Kinect sensor working and that is to download the Kinect v1.8 SDK available here;
https://www.dropbox.com/s/8z5chg39vm40f0a/KinectSDK-v1.8-Setup.exe?dl=0
This will install the Kinect Management Service. This service must be running in order for the Cubist plugin to work.
